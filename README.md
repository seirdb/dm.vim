# dm.vim

## Description

This is a Vim plugin for a custom file format with minimal syntax-highlighting functionality.

## Installation

### [Vundle][v]

```vim
Plugin 'stijngeysen/dm.vim'
```

### [vim-plug][vp]

```vim
Plug 'stijngeysen/dm.vim'
```
